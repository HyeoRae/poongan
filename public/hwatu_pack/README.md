# 고스톱(화투) 카드 에셋

업로드해 주신 화투 이미지 48장을 게임에서 바로 쓰도록 정리한 패키지입니다.
(이미지 저작권/사용 권한은 사용자 확인 완료)

## 폴더
- `cards/`     : 카드 PNG 48장 (투명 배경, 균일 316x485 캔버스, 원본 그림 그대로)
- `deck.json`  : 전체 카드 메타데이터
- `preview.png`: 월별 정렬 미리보기

## 파일 ID 규칙
`m{월2자리}_{월내순번}_{종류}`  예) `m01_1_gwang`, `m08_2_yeolggeut`
- gwang=광, yeolggeut=열끗, tti=띠, pi=피

## deck.json 필드
| 필드 | 설명 |
|------|------|
| id / file | 카드 ID, 파일 경로 |
| src | 원본 업로드 파일 번호(1~48) |
| month / flower | 월(1~12), 꽃 이름 |
| kind | gwang(광) / animal(열끗) / ribbon(띠) / junk(피) |
| name | 카드 이름 |
| ribbon_color | red / blue / rain (띠 카드일 때) |
| tags | gwang, godori, hongdan, cheongdan, chodan, ssangpi, rain ... |

## 점수 계산용 태그 정리
- **광(gwang)** 5장: 1월 학, 3월 벚꽃, 8월 달, 11월 봉황, 12월 비광
  - 3광=3점(비광 포함 시 2점), 4광=4점, 5광=15점
- **고도리(godori)** 3장: 2월 꾀꼬리, 4월 두견새, 8월 기러기 → 모으면 5점
- **홍단(hongdan)** 3장(1·2·3월) / **청단(cheongdan)** 3장(6·9·10월) / **초단(chodan)** 3장(4·5·7월) → 각 3점
- **쌍피(ssangpi)** 3장: 9월 술잔(열끗 겸용), 11월 쌍피, 12월 쌍피
- 피 10장 = 1점 (이후 1장당 +1점), 쌍피는 2장으로 계산

## 참고
- 카드 순서는 원본이 역순(1~4=12월 → 45~48=1월)이라, ID는 월 오름차순으로 다시 매겼습니다.
- **11월 쌍피**는 6·7·8번 중 빨강 영역이 있는 카드(현재 `m11_2_pi`)로 지정했습니다.
  실제 룰셋과 다르면 deck.json에서 해당 카드의 tags만 바꿔주세요(한 줄 수정).
