# 🛠️ 설정 가이드 (옵션 조정 설명서)

앱의 거의 모든 값은 코드 몇 군데만 바꾸면 조정됩니다. 아래 표에서 **"바꾸는 곳"** 과 **"적용 방법"** 을 보고 수정하세요.

## 적용 방법 3가지 (먼저 이해하기)
| 종류 | 무엇 | 적용하려면 |
|------|------|-----------|
| 🟦 코드 | 화면/연출/문구/색 (`.tsx`, `.css`) | 저장하면 끝 (개발서버는 자동반영, 배포는 다시 push) |
| 🟨 SQL | 도박 배당률 등 DB 함수 (`.sql`) | Supabase **SQL Editor**에 다시 붙여넣고 **Run** |
| 🟩 시드 | 계정/팀/일정 (`accounts.json`, `seed.ts`) | 파일 수정 후 터미널에서 `npm run seed` |

---

## 1. 🟩 친구 계정 (아이디 / 비번 / 이름)
- **바꾸는 곳**: [supabase/accounts.json](../supabase/accounts.json)
- 이름·아이디·비번을 고친 뒤 → `npm run seed` 실행
- ⚠️ 비밀번호는 **6자 이상** 권장. 아이디(username)를 바꾸면 **새 계정이 생기고 옛 계정은 남습니다**(이름/비번만 바꾸는 건 안전).

## 2. 🟩 팀 이름 / 팀 색깔
- **바꾸는 곳**: [supabase/seed.ts](../supabase/seed.ts) 의 `seedTeams()` 함수
  ```ts
  { id: 1, name: "통영팀", color: "#3b82f6" },  // 파랑
  { id: 2, name: "거제팀", color: "#ef4444" },  // 빨강
  ```
- `name`(이름), `color`(색, [HEX 색상코드](https://htmlcolorcodes.com/)) 수정 → `npm run seed`
- 색은 대시보드 막대·팀 카드·배정식 슬롯에 모두 자동 반영됩니다.

## 3. 🟩 여행 일정
- **바꾸는 곳**: [supabase/seed.ts](../supabase/seed.ts) 의 `seedSchedule()` 함수 (day/시간/제목/장소/설명)
- ⚠️ 시드는 **일정이 비어있을 때만** 넣습니다. 이미 들어간 걸 고치려면:
  - **방법A (쉬움)**: Supabase → **Table Editor → `schedule`** 에서 직접 행을 추가/수정/삭제
  - **방법B**: Table Editor에서 `schedule` 행 전체 삭제 후 → `npm run seed` 다시 실행
- `sort_order` 숫자가 작을수록 위에 표시됩니다.

## 4. 🟦 도박 배당률 / 확률
- **바꾸는 곳**: [supabase/migrations/0002_gamble.sql](../supabase/migrations/0002_gamble.sql)
  - **동전던지기**: `gamble_coinflip` 안의 `random() < 0.5`(당첨 확률), `p_bet * 2`(당첨 배수)
  - **주사위**: `gamble_dice` 안의 `p_bet * 6`(당첨 배수) — 6배 지급
- 수정 후 → 해당 함수 SQL을 **SQL Editor에 다시 붙여넣고 Run** (함수는 덮어쓰기됨)

## 5. 🎬 팀 배정식 연출 (속도 / 효과)
- **바꾸는 곳**: [components/DrawCeremony.tsx](../components/DrawCeremony.tsx) 상단
  ```ts
  const SPIN_MS = 1800;   // 슬롯머신 도는 시간(ms) — 길수록 긴장감 ↑
  const SETTLE_MS = 800;  // 착지 후 다음으로 넘어가는 텀(ms)
  ```
- 컨페티 양: `burst()`의 `particleCount: 60`, 피날레는 `finale()` 함수
- 효과음 톤: `beep(220, ...)`(도는 소리), `beep(660, ...)`(착지 소리) 숫자가 음 높이

## 6. 🟦 풍산토큰 운영
- 초기 풍산토큰은 **0**에서 시작. 지급/차감은 **앱 안에서**: 관리자 → `⚙️ 관리` 탭 → "개인/팀 풍산토큰 지급·차감"
- 차감은 **음수**로 입력 (예: `-100`)
- 게임/미션 보상은 전부 이 패널로 수동 지급하면 됩니다 (사유 입력 → 내역에 기록됨)

## 7. 🟦 앱 제목 / 로그인 화면 문구
- 상단 탭 제목: [app/layout.tsx](../app/layout.tsx) 의 `metadata.title`
- 로그인 화면 큰 글씨/부제: [app/login/page.tsx](../app/login/page.tsx)

## 8. 🟦 테마 색 (골드색 / 배경)
- **바꾸는 곳**: [app/globals.css](../app/globals.css) 의 `:root`
  ```css
  --background: #0b0b12;  /* 배경 */
  --gold: #f5c542;        /* 포인트 골드색 */
  --card: #16161f;        /* 카드 배경 */
  ```

## 9. 🟦 하단 메뉴 (탭)
- **바꾸는 곳**: [components/BottomNav.tsx](../components/BottomNav.tsx) 의 `items` 배열 (아이콘/이름/링크)

## 자주 쓰는 명령어
```bash
npm run dev     # 로컬 개발 서버 (http://localhost:3000)
npm run seed    # 계정/팀/일정 다시 심기 (accounts.json·seed.ts 반영)
npm run build   # 배포 전 오류 점검
```

- 🟦 코드·🟨 SQL 변경은 **GitHub에 push하면 Vercel이 자동 재배포**합니다 (배포 가이드 참고).
- 🟩 시드(계정/팀/일정)는 DB에 직접 반영되므로 배포와 무관하게 `npm run seed` 한 번이면 됩니다.
