# 변경 이력

 v1.0 : 일정 개편 · 풍산토큰 명칭 통일 · 푸시 알림 추가
일정표를 실제 여행 일정으로 교체하고 1·2·3일차 서브탭으로 분리 (진행 중/지난 일정 자동 색상 표시)
앱 전체의 "골드" 명칭을 "풍산토큰"으로 통일 (잔액 부족 알림 메시지 포함)
웹 푸시 알림 기능 추가 — 관리자가 참가자 폰으로 공지 발송 (안드로이드/아이폰 PWA 지원)
관리자 패널을 이벤트·토큰·알림·참가자 서브탭으로 정리

 v1.01 : 서비스워커 등록 오류 수정
미들웨어가 서비스워커(sw.js)를 로그인 화면으로 리다이렉트하던 문제 수정 — 푸시 알림 등록이 정상 동작

 v1.1 : 첫 접속 시 알림 설정 안내 화면 추가
로그인 후 알림이 꺼져 있으면 전체화면으로 "알림 켜기"를 먼저 권유 (미지원/아이폰 미설치 시 안내 후 건너뛰기 가능)
"나중에 하기"는 세션 단위로 처리 — 다음 로그인 때 미설정이면 다시 권유
일정표 1·3일차 문구 다듬기 (거제로 출발 / 근처 카페로 이동)

 v1.2 : 카톡 공유 미리보기 · 오프라인 대응 · 에러 화면 추가
단톡방에 앱 링크 공유 시 제목·설명·썸네일 카드가 예쁘게 뜨도록 미리보기(OG) 이미지 추가
신호 약한 여행지에서 인터넷이 끊겨도 흰 화면 대신 안내 화면을 보여주는 오프라인 대응 추가 (알림 켜지 않은 사람도 적용)
에러·없는 페이지(404)·로딩 화면을 영문 기본 화면에서 앱 톤에 맞는 한글 화면으로 교체
브라우저 탭/북마크 아이콘(favicon) 추가

 v1.3 : 일정별 미니게임(예측 배팅) · 룰렛 추가
일정마다 실제 결과에 풍산토큰을 거는 "예측 배팅" 게임 추가 — 자유 금액으로 베팅하고, 적중하면 모인 팟을 베팅한 비율대로 나눠 가집니다 (게임 탭 신설)
관리자가 일정에 게임을 만들어 열고(오픈)·마감하고·우승자를 골라 정산하는 운영 기능 추가 (관리자 패널 🎮 게임 탭)
프리셋 2종 기본 제공 — 🎣 강태공 배팅(낚시 최다 어획왕), 🛷 루지1등 배팅 (게임을 열면 그때의 참가자 명단으로 후보 자동 생성)
게임이 열리면 일정 카드에 배팅 바로가기 배지가 뜨고, 끝나면 우승자가 표시됩니다
도박(카지노)에 🎡 룰렛 추가 — 언더·오버·홀·짝 2배, 숫자 적중 시 10배
송금 화면에서 관리자·test 계정은 받는 사람 목록에서 제외 (잘못 보내는 것 방지)

 v1.4 : 🃏 섯다 실시간 멀티플레이 추가
참가자끼리 풍산토큰을 걸고 한 판 붙는 실시간 섯다 게임 추가 (하단 네비 🃏 섯다 / 도박 페이지 입장 카드)
방을 만들어 친구들이 모이면 방장이 판을 시작 — 진짜 화투 그림으로 패가 한 장씩 깔립니다
2단 베팅 방식 — 첫 장 받고 1차 베팅 → 마지막 장 받고 추가 베팅 → 쇼다운
베팅 옵션 콜·다이에 더해 삥·하프·따당·직접입력 레이즈까지 지원
족보 완비 — 광땡·땡·특수패에 잡이패(암행어사·땡잡이·멍텅구리구사 재경기)까지 적용
📖 족보표 버튼으로 내 패의 등급을 바로 확인, 상대 패는 쇼다운 전까지 비공개(블러핑 보존)
누가 자리를 비워도 30초 지나면 자동 다이로 판이 멈추지 않게 처리
관리자는 진행 중인 방을 👁 참관할 수 있고, 🔓 전체 패 보기로 모든 패를 확인 가능

 v1.5 : 🎭 팀 배정식에 역할 카드 연출 추가 · 버튼 로딩 표시
팀 배정식 마지막에 역할 카드가 양 팀으로 분배되는 모션 연출 추가 — 배정이 끝나면 각자 대시보드에서 자기 카드를 탭해 비밀 역할을 확인
도박·섯다 등에서 처리 중인 버튼에 로딩 스피너 표시 (중복 클릭·먹통 느낌 개선)

 v1.6 : 프로필 사진 등록 · 배정식에 프로필카드 연출 · 봇 제외
각자 프로필 사진을 등록하는 기능 추가 — 대시보드 "내 프로필" 카드에서 사진을 올리면 팀원 목록·팀 배정식에 얼굴로 표시됩니다
팀 배정식에서 각 참가자가 사진이 담긴 프로필 카드로 나오고, 그 위로 역할 카드가 한 장씩 안착하는 연출로 변경 (카드가 화면 밖으로 튀던 문제 정리)
축포가 충분히 터진 뒤 역할 카드가 분배되도록 타이밍 조정 (카드가 축포에 가리던 문제 해결)
관리자 참가자 탭에 "배정식 참여/제외" 토글 추가 — 테스트·봇 계정을 배정식과 송금 대상에서 뺄 수 있습니다 (기존 test 아이디 자동 인식은 이 플래그로 대체)
팀 배정식 시작 시 관리자 본인 화면이 바로 배정식 화면으로 넘어가지 않던 문제 수정
팀 배정 중 "DELETE requires a WHERE clause" 오류로 배정식이 시작되지 않던 문제 수정

 v1.61 : 배정식 역할 카드 셔플·비행 모션 복원
중앙에서 카드를 섞은 뒤 한 장씩 각 프로필 카드로 날아가 안착하는 모션 연출 복원 (이전엔 제자리에서 나타나기만 하던 것 개선)
카드가 실제 참가자 프로필 카드 위치로 정확히 날아가도록 좌표 계산 방식으로 변경

 v1.62 : 배정식 "다음 공개" 반응 속도 개선
"다음 공개" 클릭 시 서버 응답을 기다리지 않고 즉시 연출이 시작되도록 개선 (낙관적 업데이트)
매 공개마다 대시보드 전체를 다시 불러오던 동작 제거 — 공개 진행은 실시간으로 전파되므로 불필요 (버튼 렉 해소)

 v1.63 : 배정식 카드 셔플 강화 · 3D 모션 연출
카드 덱 셔플 시간을 늘려 더 충분히 섞는 느낌으로 개선 (0.85초 → 2.8초)
셔플·분배에 입체(3D) 모션 추가 — 카드가 세워져 좌우로 흔들리며 섞이고, 날아갈 때 한 바퀴 뒤집히며 프로필 카드에 안착

 v1.7 : 배정식 역할 카드 배정 버튼 · 리플 셔플 · 문구 겹침 수정
팀 공개 완료(축포) 후 관리자가 "🎴 역할 카드 배정" 버튼을 눌러야 역할 카드가 분배되도록 단계 분리 (자동 진행 → 사회자 트리거)
덱을 화면 중앙으로 옮기고 크게 키움 — 카드가 좌·우 두 더미로 갈라져 번갈아 튕겨 오르며 섞이는 리플(riffle) 셔플 모션으로 교체
분배 중 "배정 완료!"·"분배 중" 문구가 겹치던 문제 수정 (분배 단계에선 상단 안내 문구 하나만 표시)

 v1.71 : 배정식 버튼 반응 속도 개선 · 공개 버튼 문구 수정
배정식 시작·역할 카드 배정·결과 확정·닫기 버튼에서 화면 전체 재조회를 제거 (상태는 실시간으로 전파되므로 불필요) — 버튼 누른 뒤 로딩이 길던 문제 해소
순서 공개 버튼 문구를 실제 공개 수 기준으로 표시하도록 수정 — 첫 순서만 "첫 순서 공개", 이후에는 "다음 순서 (n/N)"

 v1.72 : 팀 배정 초기화 기능 추가
관리자 이벤트 탭에 "↩️ 배정 초기화 (배정 전으로)" 버튼 추가 — 전원 팀·비밀 역할이 해제되고 배정식이 처음 상태로 돌아갑니다
초기화하면 모든 참가자 화면이 즉시 "팀 미배정"으로 바뀌어, 지난 배정 결과가 남아 보이던 문제 해결

 v1.8 : 하단 메뉴 정리 — 게임·섯다·도박을 "컨텐츠" 탭으로 통합
하단 네비게이션의 게임·섯다·도박 3개 탭을 "🎲 컨텐츠" 하나로 묶어 메뉴를 간결하게 정리
컨텐츠 탭을 누르면 미니게임·섯다·도박장을 고르는 허브 화면이 뜨고, 각 화면 상단 전환 탭으로 서로 바로 이동 가능

 v1.9 : 광대 역할 · 효과카드 뽑기 · 경제 개편(잔액 비밀·송금 수수료)
새 비밀 역할 "광대" 추가 — 우리 팀이 우승하고 그 안에서 내가 개인 꼴찌면 혼자 승리(팀이 지면 꽝). 배정식에서 팀마다 스파이 1명 + 광대 1명이 비밀 배정됨
효과카드 뽑기(가챠) 추가 — 컨텐츠 탭 "🎴 효과카드"에서 풍산토큰으로 뽑기(무료 3연차 후 뽑을수록 비싸짐, 꽝 포함). 상시 카드(이자꾼·행운의편자·큰손)와 1회용 카드(몰빵권·방탄조끼·재도전·관심법·무료송금·흥신소) 9종
개인 보유 토큰이 비밀로 전환 — 대시보드는 이제 팀 합산 점수만 공개하고 개인 잔액은 본인만 볼 수 있음
송금 수수료 20% 도입 — 받는 사람은 80%만 수령(효과카드로 면제·할인 가능). 개인·전체 누적 전적(획득·송금·도박 등)이 기록되어 흥신소 카드로 열람 가능
관리자: 효과카드 선물 기능 추가

 v1.91 : 관리자 참가자 거래내역 조회
관리자가 지갑 탭에서 참가자를 선택해 그 사람의 송금·획득 등 모든 토큰 내역을 확인할 수 있는 기능 추가

 v1.92 : 관리자 거래내역을 전체 타임라인 + 참가자 필터로 개선
관리자 지갑 탭이 기본으로 전체 참가자의 거래 타임라인(누구 내역인지 이름 표시)을 최신순으로 보여줌
드롭다운에서 특정 참가자를 골라 그 사람 내역만 필터링 가능

 v1.93 : 효과카드 아이콘 변경 · 뽑기 무한로딩 수정
효과카드 뽑기 아이콘을 화투(🎴)에서 조커카드(🃏)로 변경 (섯다는 화투 게임이라 🎴로 정리)
뽑기 버튼이 간헐적으로 무한로딩에 걸리던 문제 수정 — 결과를 즉시 표시하고 잔액·보유카드 갱신은 백그라운드로 분리
뽑기 연출 대기 시간을 900ms→600ms로 줄여 체감 로딩 단축

 v1.94 : 여행 일정 최신화 · 숙소 주소 · 카카오맵 연결
일정을 최신 계획에 맞게 갱신 (1일차 서프앤 윤돌섬 패들보트+스노클링 추가, 2일차 온천·통영 이동으로 정리)
1일차·2일차 숙소 주소를 일정에 반영 (거제시 남부면 다대2길 66 / 통영 제이힐펜션)
일정 항목의 📍위치를 탭하면 카카오맵에서 바로 검색되어 지도가 열리도록 링크 연결
※ 일정 반영은 서버에서 npm run seed:schedule 재시드 필요

 v1.95 : 일정 위치를 정확한 주소로 보정
서프앤 윤돌섬·1일차 숙소·선상낚시 선착장·온천·통영 숙소 위치를 문서의 정확한 도로명/지번 주소로 교체 (카카오맵 검색 정확도 향상)

 v2.05 : 벌칙 옷 랜덤 뽑기 세리머니 추가
관리자가 벌칙 옷(🍌바나나·🤡광대·🍄마리오·🎉파티)을 걸고 참가자 전원 대상으로 실시간 랜덤 뽑기를 진행하는 기능 신설
뽑기 연출 3종 지원 — 🏁동물 달리기(대기실에서 선착순으로 동물 선택)·🔮구슬 레이스(결정론 물리엔진)·🎰룰렛 회전
모든 폰이 같은 seed로 동일한 애니메이션을 재생하고, 당첨자는 서버가 winner_index로 확정해 조작 불가
당첨 이력을 기록해 누가 어떤 벌칙에 걸렸는지 확인 가능 (penalty_picks)
