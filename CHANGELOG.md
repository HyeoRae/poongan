# 변경 이력

 v1.0 : 일정 개편 · 풍산토큰 명칭 통일 · 푸시 알림 추가
일정표를 실제 여행 일정으로 교체하고 1·2·3일차 서브탭으로 분리 (진행 중/지난 일정 자동 색상 표시)
앱 전체의 "골드" 명칭을 "풍산토큰"으로 통일 (잔액 부족 알림 메시지 포함)
웹 푸시 알림 기능 추가 — 관리자가 참가자 폰으로 공지 발송 (안드로이드/아이폰 PWA 지원)
관리자 패널을 이벤트·토큰·알림·참가자 서브탭으로 정리

 v1.01 : 서비스워커 등록 오류 수정
미들웨어가 서비스워커(sw.js)를 로그인 화면으로 리다이렉트하던 문제 수정 — 푸시 알림 등록이 정상 동작

 v1.1 : 첫 접속 시 알림 설정 안내 화면 추가
로그인 후 알림이 꺼져 있으면 전체화면으로 "알림 켜기"를 먼저 권유 (미지원/아이폰 미설치 시 안내 후 건너뛰기 가능)
"나중에 하기"는 세션 단위로 처리 — 다음 로그인 때 미설정이면 다시 권유
일정표 1·3일차 문구 다듬기 (거제로 출발 / 근처 카페로 이동)

 v1.2 : 카톡 공유 미리보기 · 오프라인 대응 · 에러 화면 추가
단톡방에 앱 링크 공유 시 제목·설명·썸네일 카드가 예쁘게 뜨도록 미리보기(OG) 이미지 추가
신호 약한 여행지에서 인터넷이 끊겨도 흰 화면 대신 안내 화면을 보여주는 오프라인 대응 추가 (알림 켜지 않은 사람도 적용)
에러·없는 페이지(404)·로딩 화면을 영문 기본 화면에서 앱 톤에 맞는 한글 화면으로 교체
브라우저 탭/북마크 아이콘(favicon) 추가

 v1.3 : 일정별 미니게임(예측 배팅) · 룰렛 추가
일정마다 실제 결과에 풍산토큰을 거는 "예측 배팅" 게임 추가 — 자유 금액으로 베팅하고, 적중하면 모인 팟을 베팅한 비율대로 나눠 가집니다 (게임 탭 신설)
관리자가 일정에 게임을 만들어 열고(오픈)·마감하고·우승자를 골라 정산하는 운영 기능 추가 (관리자 패널 🎮 게임 탭)
프리셋 2종 기본 제공 — 🎣 강태공 배팅(낚시 최다 어획왕), 🛷 루지1등 배팅 (게임을 열면 그때의 참가자 명단으로 후보 자동 생성)
게임이 열리면 일정 카드에 배팅 바로가기 배지가 뜨고, 끝나면 우승자가 표시됩니다
도박(카지노)에 🎡 룰렛 추가 — 언더·오버·홀·짝 2배, 숫자 적중 시 10배
송금 화면에서 관리자·test 계정은 받는 사람 목록에서 제외 (잘못 보내는 것 방지)

 v1.4 : 🃏 섯다 실시간 멀티플레이 추가
참가자끼리 풍산토큰을 걸고 한 판 붙는 실시간 섯다 게임 추가 (하단 네비 🃏 섯다 / 도박 페이지 입장 카드)
방을 만들어 친구들이 모이면 방장이 판을 시작 — 진짜 화투 그림으로 패가 한 장씩 깔립니다
2단 베팅 방식 — 첫 장 받고 1차 베팅 → 마지막 장 받고 추가 베팅 → 쇼다운
베팅 옵션 콜·다이에 더해 삥·하프·따당·직접입력 레이즈까지 지원
족보 완비 — 광땡·땡·특수패에 잡이패(암행어사·땡잡이·멍텅구리구사 재경기)까지 적용
📖 족보표 버튼으로 내 패의 등급을 바로 확인, 상대 패는 쇼다운 전까지 비공개(블러핑 보존)
누가 자리를 비워도 30초 지나면 자동 다이로 판이 멈추지 않게 처리
관리자는 진행 중인 방을 👁 참관할 수 있고, 🔓 전체 패 보기로 모든 패를 확인 가능

 v1.5 : 🎭 팀 배정식에 역할 카드 연출 추가 · 버튼 로딩 표시
팀 배정식 마지막에 역할 카드가 양 팀으로 분배되는 모션 연출 추가 — 배정이 끝나면 각자 대시보드에서 자기 카드를 탭해 비밀 역할을 확인
도박·섯다 등에서 처리 중인 버튼에 로딩 스피너 표시 (중복 클릭·먹통 느낌 개선)

 v1.6 : 프로필 사진 등록 · 배정식에 프로필카드 연출 · 봇 제외
각자 프로필 사진을 등록하는 기능 추가 — 대시보드 "내 프로필" 카드에서 사진을 올리면 팀원 목록·팀 배정식에 얼굴로 표시됩니다
팀 배정식에서 각 참가자가 사진이 담긴 프로필 카드로 나오고, 그 위로 역할 카드가 한 장씩 안착하는 연출로 변경 (카드가 화면 밖으로 튀던 문제 정리)
축포가 충분히 터진 뒤 역할 카드가 분배되도록 타이밍 조정 (카드가 축포에 가리던 문제 해결)
관리자 참가자 탭에 "배정식 참여/제외" 토글 추가 — 테스트·봇 계정을 배정식과 송금 대상에서 뺄 수 있습니다 (기존 test 아이디 자동 인식은 이 플래그로 대체)
팀 배정식 시작 시 관리자 본인 화면이 바로 배정식 화면으로 넘어가지 않던 문제 수정
팀 배정 중 "DELETE requires a WHERE clause" 오류로 배정식이 시작되지 않던 문제 수정
